A javascript game of Cluedo.

* [CNF - Conjunctive normal form](http://en.wikipedia.org/wiki/Conjunctive_normal_form) for AI
* [Bitwise operation](http://en.wikipedia.org/wiki/Bitwise_operation) to deal with the cards.
